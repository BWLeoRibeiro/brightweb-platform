# Codex Token Platform Smoke

Project scaffolded with create-bw-app.

## Local setup

1. Review `.env.example`, then fill `.env.local` with real service credentials.
2. Run `pnpm install` from the BrightWeb workspace root.
3. Run `pnpm --filter codex-token-platform-smoke dev`.

## Enabled modules

- crm: disabled
- projects: disabled
- admin: disabled
