# Codex Token Site Smoke

Site scaffolded with create-bw-app.

## Stack

- Next.js App Router
- Tailwind CSS v4
- Local shadcn-style component primitives

## Local setup

1. Run `pnpm install` from the BrightWeb workspace root.
2. Run `pnpm --filter codex-token-site-smoke dev`.

## Starter surfaces

- `/`

Edit `config/site.ts` to change the site name, copy, and public links.
