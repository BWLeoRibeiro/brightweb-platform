# Starter Site

This app is the starter client template and visual sandbox for `brightweb-platform`.

Use it to:

- run the shared repo on its own
- define a new client through brand, env, and module config
- preview extracted package behavior
- test auth and module wiring without touching a client project

Recommended commands from the platform repo root:

```bash
pnpm install
pnpm dev
```

Current playground routes:

- `/`
- `/bootstrap`
- `/preview/app-shell`
- `/playground/auth`
- `/playground/admin`
- `/playground/crm`
- `/playground/projects`

Starter config files:

- `config/brand.ts`
- `config/modules.ts`
- `config/env.ts`
- `.env.example`
